Supported image formats

Image formats that are supported depends on your browser but in general all modern image formats should work fine. Animated banners are not yet supported and they will be converted to static one. Output images are in resolution 500x22 pixels so optimal size for the input image is width 500px and height that can be divided evenly.
Upload image that you would like to convert into room banners.
Images are not uploaded to the server and all processing is done in the browser.

TeamSpeak 5 Channel Image Generator

Simple interactive tool for generating background images for channels in new TeamSpeak 5 beta client. Upload any image and it will be resized and split into multiple images that you can simply use as banners on your channels.
